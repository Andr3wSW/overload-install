How to Install

Step 1: Download and Extract

Download the proper version for you OS and extract the file folder

Step 2: Open Chrome Extensions

Type chrome://extensions into your search bar and press enter

Step 3: Enable Developer Mode

Find the Developer Mode toggle and click it to enter developer mode

Step 4: Click Load Unpacked

Click the Load Unpacked button in the top left corner

Step 5: Select the Folder

Double click the folder that you extracted and enjoy the extension
